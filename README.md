# adamcoronel.com

The personal website and professional portfolio of Adam Coronel.

This website is heavily inspired by [perfectmotherfuckingwebsite.com](https://perfectmotherfuckingwebsite.com/) and thus inherits the following principles:
- A website should be leightweight
- A website should be responsive
- A website should be simple

This website is hosted by [GitHub Pages](https://pages.github.com/) and uses [Jekyll](https://jekyllrb.com/) to generate the content.
